# Contributing

This is a study project, not an active open-source project. Bug reports and
suggestions are welcome. Pull requests for corrections (typos, math errors,
visualization improvements) will be reviewed.

## What's Not Being Accepted

- New deep theorem proofs (this is a stub-level project by design)
- Dependencies on large external projects
- Complete formalization work (that's the Imperial College FLT project's scope)

## Guidelines

- Keep the JSON knowledge graph structure consistent
- Changes to the verification package (`flt_verification/`) must pass 43/43 tests
- Lean files should remain proof sketches (write `sorry` for deep theorems)
- No personal information, emails, or institutional affiliations in the code
